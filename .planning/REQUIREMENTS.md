# Requirements: My Portfolio 2026

**Defined:** 2026-05-31
**Core Value:** Recruiters can quickly understand who I am and see the quality of my design and engineering through a tight, curated project showcase.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Hero

- [ ] **HERO-01**: Hero headline clearly states role, focus, and primary stack
- [ ] **HERO-02**: Hero includes primary email call-to-action above the fold

### Projects

- [ ] **PROJ-01**: Homepage showcases exactly 3 curated project cards
- [ ] **PROJ-02**: Each project card includes title, concise summary, and tech stack
- [ ] **PROJ-03**: Each project card includes live demo and GitHub links
- [ ] **PROJ-04**: Each project card includes sample images

### About & Services

- [ ] **ABOUT-01**: About section includes a 3–5 sentence background + strengths summary
- [ ] **SERV-01**: Services section lists offerings and the work I take on

### Contact & Links

- [ ] **CONT-01**: Contact section includes email CTA with mailto link
- [ ] **CONT-02**: Footer displays visible email address
- [ ] **CONT-03**: Footer includes GitHub and LinkedIn links
- [ ] **CONT-04**: Resume download link is available

### Quality & Design

- [ ] **QUAL-01**: Layout is responsive and legible on mobile and desktop
- [ ] **QUAL-02**: Performance budget met (Lighthouse mobile >= 80, LCP < 2.5s)

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Projects

- **PROJ-05**: Featured project block above the grid
- **PROJ-06**: Mini case study bullets (Problem → Approach → Result)

### About & Proof

- **ABOUT-02**: Skills list tied to projects
- **PROOF-01**: Single testimonial from credible source

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Blog or newsletter | Excluded for v1 per project constraints |
| Autoplay media or heavy animations | Hurts scan speed and performance budget |
| Skill meters / percent bars | Low credibility and wastes space |
